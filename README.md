# LED Keychain
My very own mini LED keychain!!! 
It's a cute little LED keychain that changes the animation when moved.
For example, if you pick up your keys, they will have a smiley face!
You can put this on your keys, purse, backpack, or wallet.
I hope you enjoy my first-ever hardware creation.
